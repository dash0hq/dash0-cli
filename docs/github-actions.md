# GitHub Actions

The Dash0 CLI ships two composite GitHub Actions in this repository:
* [`setup`](https://github.com/dash0hq/dash0-cli/tree/main/.github/actions/setup) let CI workflows install the CLI
* [`send-log-event`](https://github.com/dash0hq/dash0-cli/tree/main/.github/actions/send-log-event) emits deployment events without any bespoke shell scripting.
